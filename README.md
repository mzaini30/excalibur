## Setting

Upload semua SQL di folder `sql/` ke hosting.

Sesuaikan pada file `base.php`:

```php
$host = 'localhost';
$username = 'username';
$password = 'password';
$database = 'database';
```

Untuk lebih aman, ubah nama `index.php`. Misal, diubah menjadi `kucing.php`, kemudian buatlah file `index.php` yang isinya kosong.
